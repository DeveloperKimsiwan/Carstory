# 카스토리

## 소개 
**자동차에 관심있는 유저들끼리 소통하는 서비스**

### 기능
- **게시물 피드** (댓글, 좋아요, 공유)
- **필터검색**(카테고리/ 차량종류)
- **태그 검색**(피드 태그 조건 검색)
- **알림** (fcm, 알림 목록) 

### 기술스택
- **Kotlin**
- **Coroutine**
- **Firebase(Store, Storage, auth, fcm, crash, analytics, function,remote config)**
- **koin**
- **LiveData**
- **AAC ViewModel**
- **Room**

### 아키텍쳐 패턴
- **MVVM**
